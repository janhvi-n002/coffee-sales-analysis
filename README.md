This project focuses on analyzing coffee shop sales data to uncover trends, patterns, and actionable insights. Using advanced Excel features, a dynamic dashboard was created to support data-driven decision-making.
 Problem Statement
## Businesses often struggle to:
•	Identify peak sales periods 
•	Understand customer preferences 
•	Track product performance 
 Data Understanding
## The dataset includes:
•	Sales transactions 
•	Product details (category & type) 
•	Pricing and quantity 
•	Date and time of purchase 
 ## Data Preparation
Steps performed:
•	Removed duplicates 
•	Checked missing values 
•	Standardized formats (date/time) 
•	Created calculated columns 
 ## Tools & Skills Applied
•	Microsoft Excel 
•	Pivot Tables & Pivot Charts 
•	Power Pivot & DAX 
•	Data Cleaning Techniques 
•	Dashboard Design 
 ## Analysis & Findings
 1. Sales Performance
•	Monthly analysis showed fluctuations in revenue 
•	Identified highest and lowest sales periods 
 2. Peak Time Analysis
•	Maximum sales occur during specific hours 
•	Helps in optimizing staff scheduling 
 3. Product Insights
•	Few products dominate overall revenue 
•	Some items have consistently low demand 
 4. Store Comparison
•	Performance varies by location 
•	Certain stores generate higher revenue consistently 

## Dashboard Highlights
•	Clean and interactive layout 
•	Key Performance Indicators (KPIs): 
o	Total Revenue 
o	Total Orders 
o	Total Quantity Sold 
•	Visual charts: 
o	Sales trend over time 
o	Product category performance 
o	Hourly sales distribution 

 ## Business Insights
•	 Focus on top-selling products to maximize profit 
•	 Improve marketing for low-performing items 
•	 Allocate staff during peak hours 
•	 Analyze store strategies for underperforming locations 
 Future Scope
•	Build interactive dashboard in Power BI 
•	Add predictive analysis (forecasting) 
•	Automate reporting using macros 
•	Integrate real-time data
